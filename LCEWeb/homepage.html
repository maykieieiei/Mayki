What do we do?

We help beginners start their journey of learning how to code & start a Web Page.




How do we do it?

Here we introduce to you the different types of markup languages that you will use to successfully
create a web page for yourself with your own design, functions and contents.



No idea where to start?

For beginners, we recommend starting simple and easy with HTML, CSS and Java Script.




HTML, CSS and JVS (JavaScript)

If HTML, CSS and JVS was a house, HTML would be the structure, CSS would be the
design and Java Script would be the functionality, which we will discuss.



HTML

HTML (Hypertext Markup Language) is the main structure of the Website or Web Page.





CSS

CSS (Cascading Style Sheets) is the design of the Website or Web Page.






JavaScript

JavaScript is the functionality of the Website or Webpage.
